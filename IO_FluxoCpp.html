Entra e saída de dados (Fluxos de dados)



Este assunto é um bônus de tudo que abordamos até agora e servirá muito quando entrar no assunto de processamento de arquivos. Será útil para formatar os dados de forma limpa e clara para o usuário.

Entra e saído de fluxo de dados é como o programa interage com o ambiente externo (usuário). Veremos neste capítulo uma baordagem do inserção e extração de dados como já haviamos usado anteriormente cin e cout, também veremos como formatar de forma clara os dados no console (a tela que abre ao rodar o programa)

Input (Entrada de dados)


O cin é um objeto do iostream que extrai dados do teclado usando o o perador de inserção de fluxo >>
Ex:
cin >> var_int;

È possível fazer encadeamento de varias cin >> por exemplo 
                        
                            int A, B, C;
                            cout << "Digite: " << endl;
                            cin >> A >> B >> C  ;
                            cout << "[A]: " << A << endl;
                            cout << "[B]: " << B << endl;
                            cout << "[C]: " << C << endl;

Outras peculiaridades usando o operador de inserção é usar o membro de dados .get() do cin para capturar um caractere. 

                        
                            char c='\0';

                            cout >>"Digite: " >> endl;

                            c= cin.get();       

                            cout >> "digitado: " >> c >> endl;

Utilizar o cin para capturar uma cadeia de caracteres ocasiona na perda de dados se caso ter um espaço inserido. Por exemplo 

                            
                                char c[15];

                                cout >> "Digite: " >> endl;

                                cin >> c;

                                cout >> "digitado: " >> c >> endl;

Para contornar este problema de perder dados por causa do espaço você pode usar cin.get(vetor_caracteres, Tam_buffer). Por exemplo 

                            
                                int sizeBuffer=20;
                                char nome[15];
                            
                                cout >> "Digite: " >> endl;
                            
                                cin.getline(nome, sizeBuffer);
                            
                                cout >> "digitado: " >> nome >> endl;

Output (saida de dados)


Assim como temos o objeto cin que manipula o fluxo de entrada através dos membros de dados get(), getline()..entre outros. Temos também os manipuladores de fluxo de dados de saída. Eles ficilitam a organização dos dados na tela como

-Formatar a largura de campo.
-Configurar dados com precisão de casas decimais.
-Esvaziar fluxos e inserir uma nova linha do fluxo de saída. -Incluir um caracter nulo no fluxo de saída e pular espaço em branco no fluxo de entrada.

Veremos alguns desses muito usuais para facilitar na desenvoltura dos nossos programas.

Para determinar as casa de precisão de um número usamos o membro de dado setprecision() da classe padrão iomanip em conjunto com o minpulador de fluxo fixed do iostream. 

                            
                                #include < iostream >
                                using namespace std;//fixed está em iostream
                                #include < iomanip >
                                using std::setprecision;
                                
                                int main()
                                {
                                    cout << "Hello REAA!" << endl;
                                    cout << fixed << endl;
                                    double pi=3.24519966;
                                    cout << "Pi: " << setprecision(2)  << pi << endl;
                                
                                    return 0;
                                }

Podemos formatar a largura do campo, usando o membro de dado setw() para isso incluímos o iomanip e destacamos a inclusão using std::setw;

O setw() desloca desloca a ultima casa do número para a próxima casa livre, caso tiver ocupada não surgirá o efeito. Veja o exemplo. 

                            
                                #include < iostream >
                                using namespace std;//fixed está em iostream
                                #include < iomanip >
                                using std::setprecision;
                                using std::setw;
                                
                                int main()
                                {
                                    cout << "Hello REAA!" << endl;
                                
                                    int A= 123456789,
                                        B= 123456789,
                                        C= 123456789,
                                        D= 123456789,
                                        E= 123456789,
                                        F= 123456789;
                                
                                    cout << setw(9)  << A << endl;//vEJA QUE A NONA CASA ESTÁ NO MESMO LUGAR NOCONSOLE
                                    cout << setw(10) << B << endl;//AQUI JÁ COMEÇA A DESLOCAR PARA A DÉCIMA CASA
                                    cout << setw(11) << C << endl;//NATURAL
                                    cout << setw(12) << D << endl;//NATURAL
                                    cout << setw(13) << E << endl;//NATURAL
                                    cout << setw(14) << F << endl;//NATURAL
                                                                
                                    return 0;
                                }

È possível criar seu próprio monipulador de fluxo de dados, personalizar as próprias funções nos dá muitas vantagens.

Veremos que com o ostream podemos fazer isso da seguinte forma. 

                            
                                #include <iostream>
                                using namespace std;//fixed está em iostream
                                                    //ostream está em iostream
                                
                                #include <iomanip>
                                using std::setw;
                                
                                ostream &tab(ostream &manip);
                                
                                
                                int main()
                                {
                                    cout << "Hello REAA!" << endl;
                                
                                    char A='A',
                                         B='B',
                                         C='C';
                                    cout << A << tab << B << tab << C << endl;
                                
                                    return 0;
                                }
                                
                                
                                ostream &tab(ostream &manip)
                                {
                                    return manip << "\t" ; //insere a tabulação no endereço de manip e retorna a tabulação
                                    //Você pode modificar com outros caracteres
                                    //Ex: '\a que é um bip sonoro'
                                    //Ex: '\n nova linhas'
                                    //Ex: "============"
                                }
