Instruções de controle



Antes de escrever um programa de computador, é preciso entender o problema proposto, mantendo atenção especial à lógica que será utilizada para criar um mecanismo específico capaz de apresentar o resultado desejado. Frequentemente, para atingir esse objetivo, é necessário criar blocos de controle de decisão ou blocos que se repetem até que uma determinada condição seja alcançada. Esses elementos estruturais são essenciais para controlar o fluxo do programa, permitindo que este responda de maneira apropriada às diferentes situações que podem surgir durante a execução. A habilidade de utilizar adequadamente estruturas de controle, como condicionais e loops, é crucial para desenvolver programas eficientes e funcionais, garantindo que a lógica do código seja robusta e capaz de lidar com uma variedade de cenários.

Em todas as linguagens de programação, existem as instruções de controle, as quais são essenciais para direcionar o fluxo de execução do programa. São elas:

Controle de decição:
if()
if..else
if..else if..else if..else
switch..case

Controle de repetição(famosos loop's):
for()
while()
do..while()

O controle de decição mais simples e mais usado é o if(..condição..). È sem dúvida o controle mais diário que temos em nossas vidas
Ex: "Se ele não vier, então eu não vou"
      "Se estiver chovendo, então eu levo guarda-chuva"
E, assim segue o baile.

Estrutura básica do código if(..condição..). 

                    
                        #include <iostream>
                        using namespace std;
                        int main()
                        {
                            if(..coondição..)
                              ..bloco de código se caso a condição for verdadeira..
                        ..bloco de codigo, caso não seja a condição esperada ..
                                     
                                         OU
                        if(..coondição..)
                        ..bloco de código se caso a condição for verdadeira..
                        ..bloco de codigo, caso não seja a condição esperada ..
                            
                            return 0;
                        }

Na primeira forma, vemos o bloco de código abaixo do if formatado com um tab a direita e isso significa que, se a condição estabelicida como argumento dentro dos parenteses do if, for verdadeira, a primeira linha após o if será executada, senão a segunda linha será executada. E isso tanto faz para a primeira ou segunda forma representada no trecho de código acima.

A condição que é estabelicida dentro do if é sempre verdadeira, ou seja o compilador sempre vai analizar se é TRUE.

Porém! quando, há necessidade de mais de uma linha de código, caso a condição for verdadeira. Ela será delimitadade dentro de chaves como segue o exemplo abaixo. 

                    
                        #include <iostream>
                        using namespace std;
                        int main()
                        {
                            if(..coondição..)
                            {
                              ..bloco 1 de código se caso a condição for verdadeira..
                              ..bloco 2 de código se caso a condição for verdadeira..
                              ..bloco n de código se caso a condição for verdadeira..
                            }
                        ..bloco de codigo, caso não seja a condição esperada ..
                                     
                            
                            return 0;
                        }
Estrutura básica do código if-else

Existe a condição else para representar o "senão", mas para o caso acima é opcional. Já que o compilador entenderá que, se nenhuma das opções dentro do if é verdadeira, então vou executar o que está do lado de fora.

O else tem uma utilidade muito forte para exercer uma condição de default para mencionar ao usuário uma menssaem de erro. 

                    
                        #include <iostream>
                        using namespace std;
                        int main()
                        {
                            if(..condição..)
                            {
                              ..bloco 1 de código se caso a condição for verdadeira..
                              ..bloco 2 de código se caso a condição for verdadeira..
                              ..bloco n de código se caso a condição for verdadeira..
                            }
                           else
                              cout<<"Algo deu errado, jovem"<<endl;
                                     
                            
                            return 0;
                        }
Estrutura básica do código if..else if..else if..else

Quando se usa apenas um if, não tem o porque usar o else afinal seja prático e simples no seu código. Veremos agora uma cascata de if..else if..else em C++. A baixo tem um código didático, representando uma cascata de condições if-else. Começando com uma variável 'A' do tipo de dado inteiro e atribuindo um valor 4 do mesmo tipo. vamos ter em mente que:

1° O comilador sempre vai analizar o primeiro if, depois o segundo e assim por diante.
2° O compilador sempre vai esperar que a condição seja verdadeira, e não que seja falsa. por isso, se caso precisa que um bloco de código seja executado caso a seja falsa, deverá usar o operador de negação '!' (exclamação), tente você mesmo. 

                    
                        #include <iostream>
                            using namespace std;                            
                            int main()
                            {
                                cout << "Hello world!" << endl;
                                int A=4;
                            
                                if(A==4)
                                    cout<<"4"<<endl;
                             ➡ else if(A==3)
                                    cout<<"3"<<endl;
                             ➡ else if(A==2)
                                    cout<<"2"<<endl;
                             ➡ else if(A==1)
                                    cout<<"1"<<endl;
                                else
                                    cout<<"Nao foi desta vez"<<endl;
                                return 0;
                            }
Estrutura básica do código switch..case

Dentro do conceito de instruçoes de controle, temos ainda o switch-case. Ele faz um chaveamento da condição que está dentro dos parenteses do switch com os case que formam o bloco de intruções, e os break faz pular fora de todo o switch. Esperimente jogar o código abaixo no code-blocks para ver o resultado. 

                    
                        #include <iostream>
                            using namespace std;

                            int main()
                            {
                                cout << "Hello REAA!" << endl;
                                int A=0;
                            
                                cout<<"insira um valor inteiro:";
                                cin>>A;
                            
                                switch(A)
                                {
                                case 8:
                                     cout<<"Vc digitou 8"<<endl;
                                    break;
                                case 5:
                                    cout<<"Vc digitou 5"<<endl;
                                    break;
                                case 7:
                                    cout<<"Vc digitou 7"<<endl;
                                    break;
                                default:
                                    cout<<"nenhuma condicao"<<endl;
                                }
                            
                                return 0;
                            }

O mesmo pode ser feito com caracteres em vez de número:

                    
                        #include <iostream>
                            using namespace std;

                            int main()
                            {
                                cout << "Hello REAA!" << endl;
                                char A='\0';
                            
                                cout<<"insira um valor inteiro:";

                                switch(A)
                                {
                                 case 'A': 
                                 case 'a':
                                     cout<<"Vc digitou A ou a"<<endl;
                                    break;
                                 case '5':
                                    cout<<"Vc digitou 5"<<endl;
                                    break;
                                 case 'B':

                                    cout<<"Vc digitou B ou b"<<endl;
                                    break;

                                    cout<<"nenhuma condicao, entendeu a grande utilidade do dafault?"<<endl;
                                }
                            
                                return 0;
                            }
Apenas intruções de controle não deixam nosso código eficaz...laço for()

Apenas instrucoes-de-controle não basta para deixar um programa otimizados e robusto. Você precisa usar os laços de repetição para monitorar, condições até que elas sejam aceitas ou que sejam capazes de calcular n vezes a nível do clock da máquina;

Vamos começar com o laço de repetição for() que sem dúvidas, além de ser mais fácil, é com ele que entendemos melhor os outros loops.
O for é frequentemente utilizado quando o número de iterações é conhecido antecipadamente ou quando é necessário iterar sobre uma sequência específica de valores. A inicialização ocorre uma vez no início, a condição é avaliada a cada iteração, e a expressão de incremento/decremento é executada após cada iteração.

Vamos agora de aluno para aluno, examinar o laço for(). Este tipo de controle em forma de repetição carrega como argumento(o que vai dentro dos parênteses) três parâmetros(valores de entrada que definem a função) separados por ponto e virgula ";". 

                    
                        para(inicialização: Condição; passo)
                          corpo ou bloco
                        fimpara
                        
                        próxima instrução

Você pode omitir a inicialização de dentro do laço, mas deverá declarar ele fora do laço e antes dele. Lembrando que toda linguagem de programação é compilada de cima para baixo e da direita para esquerda. Então se não declarar a variavel antes, ocorrerá erro de sintaxe no código.
✔inicialização, usado para dar a condição de onde quer começar a contar ou iniciar os cálculos.
✔codição, uma condição é o que determina a essencia da do trexo de código.
✔passo, é a expressão que determina que será feito a cada passo que é repetido, normalmente é usado a expressão de incrimento ou decremento.

Você pode utilizar de muitas formas a parte de condição, podendo utiliar conceitos de relação entre dois valores ou até mesmo criar um laço infinito deixando em branco os parâmetros. 

                    
                        for(int i=0; i!=44; i++)
                        for( ; ;)
                        for(int i=100; i!=0; i--)

Apenas intruções de controle não deixam nosso código eficaz..laço while()

O laço while é outra maneira de controlar um bloco de código que precisa ser verificado até que a condição seja verdadeira ou que seja preciso calcular um valor repetidas vezes. O while é usado quando a condição para continuar o loop não é conhecida antecipadamente e pode depender de entradas do usuário ou de cálculos dentro do próprio loop. A condição é avaliada antes de cada iteração, e o loop continua enquanto a condição for verdadeira. 

                    
                        while (condição) {
                           ...código a ser repetido
                        }
                        

                        ou
                        

                        int i = 1;
                        while (i <= 5) {
                            ...Código a ser repetido
                            ++i;
                        }

Apenas intruções de controle não deixam nosso código eficaz..laço do-while();

O do-while é outra estrutura de controle de fluxo em C++ usada para implementar loops (laços de repetição). Ele é semelhante ao while, mas com uma diferença crucial: no do-while, a condição é avaliada após a execução do bloco de código, garantindo que o bloco seja executado pelo menos uma vez, independentemente da condição inicial. 

                    
                        do {
                            ..código a ser repetido
                        } while (condição);

O propósito do-while é útil quando o bloco de código precisa ser executado pelo menos uma vez, e a verificação da condição pode ser feita após a execução. Isso é especialmente útil em situações em que é necessário garantir que um bloco de código seja executado antes mesmo de verificar uma condição. exemplo de uso muito comun, é a criação de um menu, ao qual é peciso que rode ao menos uma vez antes de solicitar a opção.

Exemplo:

                    
                        #include <iostream>

                            using namespace std;
                            
                            int main()
                            {
                                cout << "Hello REAA" << endl;
                                int sair=0;
                                int op=0;
                                do
                                {
                                    cout<<"[1]opcao A"<<endl;
                                    cout<<"[2]opcao B"<<endl;
                                    cout<<"[3]opcao C"<<endl;
                                    cin>>op;
                                    system("CLS");
                                }
                                while(op!=1 && op!=2 && op!=3);
                                cout<<"vc optou por["<<op<<"]"<<endl;
                            
                                return 0;
                            }

O código acima, mostra um exemplo de um menu iterativo. Dentro do argumento while tem três condiçoes que obriga o usuário a digitar infinitamente uma das opções, fazendo uso do operador lógico &&(and) determino que a variavel op será 1, 2 ou 3. Conclusões finais

Instruções de controle em laços de repetição são essenciais para criar fluxos de execução flexíveis e dinâmicos em programas. Os laços de repetição, como for, while e do-while, permitem que um bloco de código seja executado repetidamente enquanto uma condição específica é atendida. A utilização de instruções de controle nesses laços serve a diversas finalidades:

Automatização de Tarefas:

Permite a automação de tarefas repetitivas, onde o mesmo conjunto de operações precisa ser executado várias vezes.

Manipulação de Dados em Massa:

Facilita o processamento de grandes conjuntos de dados, permitindo que operações sejam aplicadas a cada elemento do conjunto.

Iteração sobre Sequências:

Possibilita a iteração sobre sequências, como arrays ou listas, para processar cada elemento individualmente.

Implementação de Lógica de Controle:

Oferece uma maneira de implementar lógica de controle condicional dentro do laço, decidindo quando continuar ou encerrar as iterações.

Em resumo, instruções de controle com laços de repetição são fundamentais para criar programas mais flexíveis, eficientes e adaptáveis, lidando com a repetição de tarefas e respondendo dinamicamente às mudanças nas condições do programa.
