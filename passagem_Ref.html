Passagem de argumentos por referência em funções



Há três maneiras em C++ de passar argumentos para funções. È elas.
☑ Passagem por valor.
☑ Passagem por referência com argumentos de referência
☑ Passagem por referência com argumentos de ponteiro

A passagem por referência com argumentos de referência e passagem por referência com argumentos de ponteiro têm semelhanças em termos de permitir que uma função modifique diretamente o valor original de uma variável fora do seu escopo. No entanto, há diferenças importantes na sintaxe e no comportamento associado a cada uma. Antes de verificar o programa a baixo, leia atentamente o que diz a respeito das diferenças entre elas.

Antes disso, temos a passagem por valor, que refere-se à prática de passar argumentos para funções através de cópias dos valores originais. ou seja, a função recebe a cópia do(s) valor(es) e a função trabalha com a variável lá dentro e depois retorna um resultado sem alterar o valor original da variável que foi passada como argumento.

Já a passagem por referência com argumento de ponteiro, é passado o argumento como endereço(&) de onde está o valor e alterado diretamente na variável.
Ex:
cubebyReferenceByPoiter(&num);

E a Passagem por referência com argumento de referência, é passado o argumento normalmente ou seja sem (*) e (&).
Ex:
cubebyReferenceByAddres(numero);

Ambas as abordagens têm suas aplicações e escolher entre elas depende dos requisitos específicos do código e do estilo de programação preferido. Em geral, a passagem por referência com argumentos de referência é preferida quando se trabalha com objetos existentes e não nulos. A passagem por referência com argumentos de ponteiro pode ser útil quando a nulidade é uma consideração importante ou quando a manipulação de ponteiros é desejada. Teste o programa a baixo para melhor entendimento, você pode usar qualquer editor de programa, particularmente eu uso o cod::blocks por facilitar a criação de arquivos header, como veremos mais adiante. 

                    
                        #include <iostream>
                            using namespace std;
                            
                            //========================================
                            //--Protótipo da(s) funções--

                            int cubebyValue(int);//Passagem por valor
                            void cubebyReferenceByAddres(int&);//Passagem por eferencia
                            void cubebyReferenceByPoiter(int*);

                            //========================================
                            //--Corpo do programa--

                            int main()
                            {
                                cout << "Hello REAA" << endl;
                                int number   =5,
                                    numero   =10,
                                    num      =40,
                                    auxiliar =0;
                            
                                cout<<endl;
                                cout <<"===Passagem por valor==="<<endl;
                                cout <<"O valor original de number eh "<< number;
                                auxiliar = cubebyValue(number); //passa o argumento number por valor;
                                cout<< "\nO valor de number continua o mesmo " << number << endl;
                                cout<< "O valor que retorna eh " << auxiliar << endl;
                                cout<<endl;
                            
                                cout <<"===Passagem por referencia com argumentos de referencia===" << endl;
                                cout <<"O valor original de numero eh "<<numero<<endl;
                                cubebyReferenceByAddres(numero);//Não retorna nada, pois ele altera diretamente 
 
                                                                //no endereço da variavel numero
                                cout <<"O novo valor de numero eh "<<numero<<endl;
                            
                                cout<<endl;
                            
                                cout <<"===Passagem por referencia com argumentos de ponteiro===" << endl;
                                cout <<"O valor original de num eh " << num <<endl;
                                cubebyReferenceByPoiter(&num);
                                cout <<"o novo valor de num eh " << num <<endl;
                            
                                return 0;
                            
                            
                                return 0;
                            }//Fim
                            
                            
                            //========================================
                            //--Desenvolviemnto da(s) funções--

                            //cálcula e retorna  o cubo do valor inteiro
                            int cubebyValue(int number)
                            {
                                return number*number*number;
                            }//Fim da função cubValue
                            
                            void cubebyReferenceByAddres(int &n)
                            {
                                n=(n)*(n)*(n);// o mesmo que n=n*2;
                            }//Fim da função cubebyReferenceByAddres
                            
                            void cubebyReferenceByPoiter(int *n)
                            {
                                *n = (*n)*(*n)*(*n);
                            }//Fim da função cubebyReferenceByPoiter
