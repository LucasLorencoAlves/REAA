Sobrecarga (de funções e de operadores)


...Sobrecarga de funções



Sobre carga de funções é uma alternativa a templates a diferença entre eles é que sobrecargas de funçoes é limitada ao uso de tipos de dados ou tipos de parâmetros. A sobre carga de funções exige que para cada possíveis tipos de dados de entrada seja uma função.

Diferentemente de um template que com apenas uma função template é possível combinar tipos de dados de entrada. Mas não deixa de ser um ótimo recurso ou técnica de programação. Veja o exemplo a baixo. 

                    
                        #include <iostream>
                        using namespace std;

                        
                        int      soma(int, int);      //Função soma que recebe dois inteiros
                        double   soma(double, double);//Função soma que recebe dois decimais
                        int      soma(int, int, int); //Função soma que recebe três inteiros


                        int main()
                        {
                            cout << "Hello REAA!"<< endl;
                            cout<<"soma com 2 int:"<<soma(2, 3)    <<endl;//Retorna a soma de dois inteiros
                            cout<<"soma com 3 int:"<<soma(2, 3, 4) <<endl;//Retorna a soma de três inteiros
                            cout<<"soma com 2 double:"<<soma(8.2, 1.9)<<endl;//Retorna a soma de dois decimais

                            return 0;
                        }

                        //sobrecarga de dois inteiros
                        int      soma(int A, int B)
                        {
                            return A+B;
                        }
                        //sobrecarga de dois double

                        double   soma(double A, double B)
                        {
                            return A+B;
                        }
                        //sobrecarga de três inteiros

                        int      soma(int A, int B, int C)
                        {
                            return A+B+C;
                        }

Sobre carga de funções é isso, mais um recurso da linguagem C++ que permite a reutilização de código

...Sobrecarga de operadores



Sobrecargas de operadores em C++, é um recurso adicional para expandir a linguagem, facilitar a leitura e operação de objetos criados por você. Podemos dizer que é mais cosmético trabalhar com sobrecarga de operadores, mas facilita muito em simplificar e gerenciar memória.

De forma mais simplificada, sobrecarregar um operador, é usar um operador que tem a função nativa de operar com variáveis simples e fazer elel operar com estruturas ou objetos. Por exeplo, è possível multiplicar duas variáveis A e B do tipo inteiro ou double. Mas! não dá para multiplicar dois objetos espacias(que recebem três parâmetros).

Por isso! Sobrecarregar o operador (*) para multiplicar dois objetos é uma prática que simplifica significativamente a construção, reutilização e manutenção do programa. Essa abordagem proporciona uma expressividade aprimorada ao código, tornando-o mais intuitivo e conciso. Abaixo estão exemplos simples que evidenciam a utilidade dessa técnica. Eu como estudante encorrajo a analisar, modificar e estudar esses e outros exemplos para aprofundar a compreensão sobre como a sobrecarga de operadores pode ser aplicada de maneira eficaz. Essa prática não apenas facilita o desenvolvimento, mas também contribui para interfaces mais claras e código mais legível, resultando em um programa mais robusto e de fácil manutenção.
