A pesquisa binária é um jeito esperto e eficiente de procurar um item dentro de uma lista que já está organizada. Imagina que você tem uma lista de números ou de qualquer outra coisa que já está em ordem crescente ou decrescente. Em vez de procurar item por item, você vai cortando a lista pela metade até encontrar o que está procurando. É tipo um "Vamo logo pra parte que interessa!" da programação.

Como Funciona a Pesquisa Binária


Divide e Conquista:

Primeiro, você pega o meio da lista. Se o número que você está procurando for menor que o do meio, você ignora a parte da frente e foca só na parte de trás (ou vice-versa).

Repetir até Encontrar:

Vai repetindo isso, dividindo a lista ao meio, até achar o número que você quer ou até sobrar nada pra procurar (aí você sabe que o número não está na lista).

A baixo está um código simples para similar a divisão susseciva até encontrar o valor da chave. Dentro do comando while está a as três condições que rege o algoritimo, se a chave é maior que o centro, então o inicio do vetor é deslocado para a direita se caso a chave é menor então o fim é deslocado para a esquerda. Fazendo assim até que o centro seja igual a chave pesquisada. 

                        
                            #include <iostream>
        
                                using namespace std;
                                
                                int main()
                                {
                                    cout << "Hello world!" << endl;
                                
                                    int vetor[]={14, 20, 32, 47, 55, 63, 71, 89};
                                
                                    for(int i=0;  i<7; i++)
                                    {
                                
                                        cout << " >> "  << i << endl;
                                    }
                                
                                    int indice=-1,
                                        Fim_vetor=8,
                                        Ini_Vetor=0,
                                        Centro=(Fim_vetor - Ini_Vetor)/2,
                                        chave=89;
                                
                                
                                    while( (Ini_Vetor <= Fim_vetor) && indice ==-1 )
                                    {
                                        if( chave == vetor[Centro])
                                            indice=Centro;
                                        else if(chave <= vetor[Centro])
                                            Fim_vetor=Centro-1;
                                        else
                                            Ini_Vetor=Centro+1;
                                
                                    Centro=(Ini_Vetor + Fim_vetor)/2;
                                
                                    }
                                
                                    cout<<"Encontrado: " << indice << " na posicao " << endl;
                                
                                
                                    return 0;
                                }
