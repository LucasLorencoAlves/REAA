Templates(de funções e templates de classe)



Templates são uma alternativa à sobrecarga de funções, quando envolve lógica de programação idênticas para varios tipos de entrada de dados.

Quando você cria uma função normal em C++, ela é específica para um tipo de dado. Por exemplo, uma função que soma dois números inteiros não funcionará diretamente para números decimais. Aqui é onde entram os templates de funções, e até mesmo templates de classe que veremos no decorrer desta página.

Quando você cria uma função normal em C++, ela é específica para um tipo de dado. Por exemplo, uma função que soma dois números inteiros não funcionará diretamente para números decimais. Aqui é onde entram os templates de funções.

Com templates de funções, você pode criar uma função que funciona para diferentes tipos de dados. Por exemplo: 


                        
                        #include <iostream>
                        using namespace std;
                                
                        template<typename   NewType>
                        void  imprimi( NewType numero1,  NewType numero2,  NewType numero3,  NewType numero4);
                                
                                
                        template<typename   NewType>
                        void  imprimi( NewType numero1,  NewType numero2,  NewType numero3);
                        
                                
                        template<typename   NewType> 
                        void  imprimi( NewType numero1,  NewType numero2);
                                
                                
                        template<typename   NewType>
                        void  imprimi( NewType numero1);
                                
                        int main()
                        {
                            cout  <<  "Hello REAA!"  <<  endl;
                             imprimi(99, 44, 3, 7);
                             imprimi(1, 8, 37);
                             imprimi(17, 66);
                             imprimi(1025);
                                
                                
                                
                                
                            return 0;
                        }
                                
                                
                                
                        template<typename   NewType>
                        void  imprimi( NewType numero1,  NewType numero2,  NewType numero3,  NewType numero4)
                        {
                            cout << "imprime: " << numero1 << '-' << numero2 << '-' << numero3 << '-' << numero4 << endl;
                        }
                                
                                
                        template<typename   NewType>
                        void  imprimi( NewType numero1,  NewType numero2,  NewType numero3)
                        {
                            cout << "imprime: " << numero1 << '-' << numero2 << '-' << numero3 << endl;
                        }
                                
                                
                        template<typename   NewType>  
                        void  imprimi( NewType numero1,  NewType numero2)
                        {
                            cout << "imprime: " << numero1 << '-' << numero2 << endl;
                        }
                                
                                
                        template<typename   NewType> 
                        void  imprimi( NewType numero1)
                        {
                            cout << "imprime: " << numero1 << endl;
                        }

Veja a sintaxe da construção de um template de funções, antes do protótipo da função e até mesmo antes da declaração do desenvolvimento da dela, é declarado a palavra reservada template onde, template indica que a função será um template, typename será um corringa para um tipo de dado, isso significa que o usuário irá colocar algum tipo de dados (int, double, string.. ) e o compilador vai identificar qual tipo de dado e substituir o typename em tempo de compilação. A variável NewType é um argumento qualquer e eu poderia ter colocar qualquer coisa.

Template não tem muito mistério, mas é importante fazer leitura de algumas obras para te conhecimento sobre boas práticas e dicas de uso em projetos reais.



Templates(de classes)



Templates de classe tem o mesmo propósito de template de funções, não precisar rescrever varias classes com o mesmo algoritimo mas com diferentes entradas de tipos de dados.

Analise o código abaixo. cada objeto de mesma classe imprimi diferentes tipos de dados inclusive caracteres ou strings. Determino uma quantidade X para o vetor de dados e um tipo para cada template. E finalmente chamo o objeto que imrpimi os dados. 

    
                        #include <iostream>
                        using namespace std;
                        #include <string>
                        using std::string;
                        
                        
                        //=================================
                        //--Clsse--
                        
                        template<typename newType, unsigned short U>
                        class minhaClasse
                        {
                        
                        public:
                            minhaClasse();
                            void gravVetor();
                            void mostraVetor();
                        
                        private:
                        
                            newType vetor[10];
                        
                        
                        };
                        
                        int main()
                        {
                            cout << "Hello REAA!" << endl;
                        
                            //================================
                            //--com int--
                            minhaClasse<int, 2> m1;
                            cout<<"inserindo valor do tipo int\n";
                             //m1.mostraVetor();
                            m1.gravVetor();
                            m1.mostraVetor();
                            
                            //================================
                            //--com double--
                            cout<<"inserindo valor do tipo double\n";
                            minhaClasse<double, 2> m2;
                             //m2.mostraVetor();
                            m2.gravVetor();
                            m2.mostraVetor();
                            
                            //================================
                            //--com char--
                            cout<<"inserindo valor do tipo char\n";
                            minhaClasse<char, 2> m3;
                                 //m3.mostraVetor();
                            m3.gravVetor();
                            m3.mostraVetor();
                        
                            
                            //================================
                            //--com string--
                            cout<<"inserindo valor do tipo string\n";
                            minhaClasse<string, 2> m4;
                            //m4.mostraVetor();
                            m4.gravVetor();
                            m4.mostraVetor();
                        
                        
                        
                            return 0;
                        }
                        
                        
                        //=================================
                        //--Desenvolvimento de funções--
                        
                        template<typename newType, unsigned short U>   
                        minhaClasse<newType, U>::minhaClasse()
                        {
                            for(int i=0; i<U; i++)
                            {
                                vetor[i]='\0';
                            }
                        
                        
                        }
                        
                        
                        
                        
                        template<typename newType, unsigned short U> 
                        void minhaClasse<newType, U>::gravVetor()
                        {
                            for(unsigned short i=0; i<U; i++)
                            {
                                cout<<"insira um numero: "<< '[' << i<<'/'<<U<<']';
                                cin>>vetor[i];
                            }
                        }
                        
                        
                        template<typename newType, unsigned short U>  
                        void minhaClasse<newType, U>::mostraVetor()
                        {
                            for(int i=0; i<U; i++)
                            {
                                cout<<":"<<vetor[i]<<endl;
                            }
                        
                        }

Este é um exemplo bastente simples para começar a entender sobre Templates de funções e classes muito útil em sistemas embarcados.
